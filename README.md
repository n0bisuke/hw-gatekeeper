# GATEKEEPER - 宿題管理自動化システム

Notionの宿題データベースを定期的に監視し、提出期限内かつ内容に不備がないものだけを転記先スプレッドシートへ転記する「門番」システム。不備や期限外の提出は自動的に却下し、Discord/Teamsへ理由を通知する。

## セットアップ

```bash
cp .env.example .env
# .envに各種キーを設定
npm ci --ignore-scripts
node index.js
```
